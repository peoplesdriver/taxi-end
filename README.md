# taxi-end
Taxi End Github Page


# SETUP
1-Clone or Download this repo

2-Run in "compser install" command line
```
#!terminal

compser install

```

3-Run "php artisan migrate"
```
#!terminal

php artisan migrate

```

4-Run "php artisan serve" in command line to start the development server
```
#!terminal

php artisan serve

```
